파이썬을 위한 다양한 개발환경

## 가상환경 생성(venv 활용)
 ` > python -m venv jnum `   
 ` > jnum\scripts\activate `  
 ` > pip install numpy jupyter `  
 ` > pip show numpy jupyter `  
 ` > deactivate `

## 가상환경 생성(conda 활용)
 ` > conda create -n cjmat matplotlib jupyter ` 
 ` > conda activate cjmat `
 ` > conda install pandas tensorflow `
 ` > conda list pandas `
 ` > conda list jupyter `
 ` > conda deactivate `
